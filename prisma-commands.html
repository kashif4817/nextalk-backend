Prisma Commands Cheat Sheet

NexTalk Backend — Prisma 7 + Supabase PostgreSQL

⚙ Schema & Database

CommandDescription
npx prisma generate Generates the Prisma Client from your schema
npx prisma db push prototyping Pushes schema to DB without creating a migration file
npx prisma migrate dev dev Creates a migration file and applies it
npx prisma migrate deploy prod Applies pending migrations (use in production)
npx prisma migrate reset dev Drops the DB and re-applies all migrations from scratch

🔍 Debugging & Exploration

CommandDescription
npx prisma studio Opens a visual GUI to browse and edit your data
npx prisma validate Checks your schema for errors
npx prisma format Auto-formats your schema file
npx prisma db pull Pulls existing DB schema into your schema.prisma (introspection)

⚡ package.json Shortcuts

ShortcutRunsDescription
npm run db:generate npx prisma generate Generate Prisma Client
npm run db:migrate npx prisma migrate dev Create & apply migration
npm run db:push npx prisma db push Push schema without migration

🚀 Typical Workflow

1. Edit schema.prisma → 2. npx prisma migrate dev --name "describe_change" → 3. npx prisma generate → 4. Use updated client in your code
